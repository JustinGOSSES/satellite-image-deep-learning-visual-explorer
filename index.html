This webpage was built from a template that automates the process of grabing all the GitHub repos from an Awesome List in one markdown file, harvesting details about those repos using the GitHub API, and creating visualizations that make it easier to understand the trends & connections between repositories.

GitHub repositories were programmatically sourced from the {{ site.awesome_list_repo_name }} awesome list.

The main point of the website is the visualizations, which you can start to explore from the explore section! Note that the menu to dependencies visualization page does not work great on some phones yet.

See all of the parsed repositories in the catalog view.
